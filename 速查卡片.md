# 博客速查卡片

## 新建文章
```bash
cd ~/Desktop/blog
hugo new posts/YYYY-MM-DD-英文短名.md
```

## 文章头部模板
```yaml
---
title: "文章标题"
slug: "english-slug"
description: "一句话简介"
draft: false
---

正文写这里...
```

## 图片
- 放到 `static/images/` 
- 文章里引用：`![](/images/xxx.png)`

## 本地预览
```bash
cd ~/Desktop/blog && hugo server -D
# 浏览器打开 http://localhost:1313
```

## 发布
```bash
cd ~/Desktop/blog
git add -A
git commit -m "新文章：xxx"
git push
```
